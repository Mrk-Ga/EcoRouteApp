# EcoRouteApp

EcoRouteApp is an Android application that provides real-time air quality monitoring and route tracking.

## Features

*   **User Authentication**: Secure login and registration for users.
*   **Air Quality Monitoring**: View real-time air quality data.
*   **Route History**: Keep track of your past routes and view their details.
*   **Sensor Reporting**: Report faulty or inaccurate sensors.
*   **Admin Panel**: A dedicated section for administrators to manage users and stations.
*   **Settings**: Customize your application experience.

## Technologies Used

*   [Kotlin](https://kotlinlang.org/) as the primary programming language.
*   [Jetpack Compose](https://developer.android.com/jetpack/compose) for building the user interface.
*   [Jetpack Navigation](https://developer.android.com/jetpack/compose/navigation) for navigating between screens.
*   [Retrofit](https://square.github.io/retrofit/) for handling network requests.
*   [Google Play Services Location](https://developers.google.com/android/guides/location-overview) for accessing device location.
*   [Kotlin Coroutines](https://kotlinlang.org/docs/coroutines-overview.html) for asynchronous operations.
*   [Android ViewModel](https://developer.android.com/topic/libraries/architecture/viewmodel) for managing UI-related data.

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

*   Android Studio
*   An Android device or emulator

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username_/EcoRouteApp.git
    ```
2.  Open the project in Android Studio.
3.  Build the project and run it on an emulator or a physical device.
